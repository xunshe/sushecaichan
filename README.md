### 主要功能 
本系统适用于java毕业设计和java课程设计，基于java语言实现，基于ssm框架，主要分为三个角色，分别为管理员，宿管，学生；管理员的主要功能有：系统设置，学生管理，宿舍楼管理，财产管理，财产盘点，财产报修，报废处理等功能；学生的功能有：财产管理，财产报废管理；宿管的功能有：宿舍楼管理，财产管理，财产盘点，财产报修，报废处理等功能.
